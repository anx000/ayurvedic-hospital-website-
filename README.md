# Badarinath Ayurveda Hospital Website -https://ayurvedic-hospital-website-git-main-anx000s-projects.vercel.app

A single-page marketing site for Badarinath Ayurveda Hospital, showcasing holistic treatments, doctors, and a booking form. Deployed at [ayurvedic-hospital-website](https://ayurvedic-hospital-website-git-main-anx000s-projects.vercel.app).

## Features
- Hero section with Ayurveda-themed illustration and clear CTA buttons.
- About, Treatments, Wellness, Doctors, Testimonials, and Appointment sections.
- Responsive navigation with mobile toggle and smooth grid layouts.
- Appointment form with client-side confirmation and newsletter signup.

## Tech Stack
- **HTML5** for semantic structure and accessible sections.
- **CSS3** for modern layout (flexbox/grid), custom properties, and responsive design.
- **Vanilla JavaScript** for navigation toggle, testimonial interaction, and form feedback.

## Local Setup
1. Clone the repository or download the source.
2. Open `index.html` in your browser (double-click or serve via any static server).
3. Edit `styles.css` or `script.js` as needed; changes reflect immediately on refresh.

## Deployment
- Hosted on **Vercel**. Pushes to the `main` branch trigger automatic redeploys.
